### Movies App

## Objective

To create a simple app consuming The Movie Database's API mainly for me to practice using Jetpack
Compose.
Features:

* A home screen with a list of movies
* A details screen with details of said movies
* A search screen

## Libraries

* Coil
* Jetpack Compose
* Kotlin Coroutines
* KtLint
* Moshi
* Retrofit
